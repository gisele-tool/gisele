## Breaking changes

The following syntax nodes have been renamed:
  :varref    -> :var_ref
  :or        -> :bool_or
  :and       -> :bool_and
  :not       -> :bool_not
  :if        -> :if_st
  :else      -> :else_clause
  :elsif     -> :elsif_clause
  :while     -> :while_st
  :seq       -> :seq_st
  :par       -> :par_st
  :task_call -> :task_call_st
